# How to Add Your Images

## Step 1: Save Your Images

You need to manually save these images to the `public/images/` folder:

### Required Images:

1. **Logo** (you've shared this)
   - Save as: `public/images/logo.png` or `public/images/logo.svg`
   - This is the white Marketini logo on transparent background

2. **Hero Background** (the first hooded figure image you mentioned)
   - Save as: `public/images/hero-hooded-figure.jpg`
   - This should be the red background with "MARKETINI" text and hooded figure

3. **Philosophy Section** (the second image)
   - Save as: `public/images/secondary-figure.jpg`
   - This is the geometric red/black background with hooded figure

## Step 2: How to Save Images

### On Mac:
1. Right-click on the image file
2. Select "Save Image As..." or just drag it to the `public/images/` folder
3. Name it exactly as shown above

### On Windows:
1. Right-click the image
2. Select "Save image as..."
3. Navigate to the `public/images/` folder
4. Save with the exact filename

## Step 3: Verify

After adding images, check that your folder looks like this:

```
public/images/
├── logo.png (or logo.svg)
├── hero-hooded-figure.jpg
└── secondary-figure.jpg
```

## Step 4: Refresh Browser

Once images are in place:
1. Make sure dev server is running (`npm run dev`)
2. Refresh your browser (Cmd+R on Mac, Ctrl+R on Windows)
3. Images should now appear!

## Troubleshooting

**Images still not showing?**
- Check the filename spelling matches exactly
- Make sure images are in `public/images/` not just `public/`
- Try hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
- Check browser console for 404 errors

**Can't find the public folder?**
- It's in the root of your project: `Marketini_Website/public/images/`
- Use Finder (Mac) or File Explorer (Windows) to navigate there
